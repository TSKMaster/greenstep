# GreenStep Project Context

## Что это за проект

`GreenStep` — конкурсный MVP для экологических обращений жителей района.

Главный продуктовый сценарий:

1. пользователь входит через `magic link`
2. ставит точку на карте
3. описывает проблему и при необходимости прикладывает фото
4. отправляет обращение
5. видит обращение на карте, в списке и в своей карточке
6. администратор может обработать заявку и изменить её статус

Ключевая рамка проекта:

- рабочее ядро — это авторизация, создание обращения и цикл его обработки
- всё, что не относится к ядру, может быть demo-слоем
- demo-разделы должны выглядеть убедительно, но не должны мешать основному сценарию

## Guest-сценарий

Сейчас гостевой режим — это не пустая витрина, а ограниченный публичный доступ:

- гость видит главную страницу
- гость видит публичную карту `/map`
- гость видит реальные обращения на карте, если в Supabase включена policy на публичное чтение `reports`
- закрытые действия открывают модалку входа вместо прямого перехода

Единый текст гостевой модалки:

`Чтобы использовать данную функцию приложения, необходимо войти.`

Кнопки в модалке:

- `Позже`
- `Войти`

## Страница создания обращения

Текущий UX:

- шаг 1 — выбрать точку на карте
- шаг 2 — описать обращение
- шаг 3 — добавить фото
- шаг 4 — подтвердить и отправить

Важно:

- шаг 1 и шаг 2 обязательны
- стартового маркера на карте больше нет
- шаг 1 считается выполненным только после реального клика по карте
- координатные поля скрыты из интерфейса
- вместо них под картой показывается статус выбора точки

## Важные договорённости

- основную главную не трогаем хаотично
- крупные UI-правки делаем аккуратно, с понятной точкой отката
- после заметных изменений обновляем опорные файлы
- demo-слой не должен ломать рабочее ядро

## Что ещё остаётся в работе

- полная ручная smoke-проверка основных сценариев на desktop
- полная ручная smoke-проверка на реальном мобильном устройстве
- проверка гостевого сценария после включения публичной policy в Supabase
- постепенная вычистка старых проблем с кодировкой по всему проекту
